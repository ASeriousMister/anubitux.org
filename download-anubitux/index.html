Anu₿itux live

This is the main version of Anu₿itux! It is designed to work only in live mode, to prevent the storage of any information on disks and preserve the privacy of the users. Using it, it is mandatory to backup all the sensible information elsewhere in a proper and secure way.

Anu₿itux install

This version can be installed on devices and in virtual machines. It is intended to be used for training and to run tests on the tools. It is absolutely not recomended to use to handle sensible information (e.g. generation of new private keys).

Anu₿itux VM

This is a ready to use virtual machine, to see how Anu₿itux looks like and run some tests or trainings. It is not recomended to use it for real cases. To run it simply import it in VirtualBox. The VirtualBox Extension Pack could be mandatory to run it in a proper way.
