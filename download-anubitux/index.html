Anu₿itux 2.2 amd64

This is the main version of Anu₿itux! 
It has been designed to work in live mode, but users can also chose to install it and do some tests on the tools. Using it in live mode, it does not store any information on disks (unless desired) and preserve users's privacy. It is mandatory to backup all information elsewhere properly and securely.

 

Anu₿itux VM for VirtualBox

This is a ready to use virtual machine to see what Anu₿itux looks like and run some tests or training. It is not yet recommended to use it for real cases. To run it, simply import it into VirtualBox. The VirtualBox Extension Pack could be mandatory to run properly.
