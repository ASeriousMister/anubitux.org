Anu₿itux live

This is the main version of Anu₿itux! 
It has been designed to work only in live mode to prevent storing user information on disks and preserve users' privacy. Using it, it is mandatory to backup all information elsewhere properly and securely.

   

Anu₿itux install

This version can be installed on devices and on virtual machines. It is intended to be used for training and to run tests on the tools. It is absolutely not recommended to use to handle sensitive information (e.g. generation of new private keys).

   

Anu₿itux VM

This is ready to use a virtual machine to see how Anu₿itux looks like and run some tests or training. It is not yet recommended to use it for real cases. To run it, simply import it into VirtualBox. The VirtualBox Extension Pack could be mandatory to run in a proper way.
