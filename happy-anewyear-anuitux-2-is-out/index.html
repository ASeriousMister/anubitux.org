Happy AnewByear! AnuBitux 2 is out!

January 1, 2023 AnuBitux

Great news! AnuBitux 2 is out! It is now based on Debian 11 bullseye and has a completely new user interface based on Cinnamon!

Now user can chose at the boot if they want to run AnuBitux in live mode, the main purpose of this project, or install it to train using its tools.

It also has a lot of new tools and updates! Download it and check it out!
