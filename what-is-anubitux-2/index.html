What is AnuBitux?

January 1, 2023 AnuBitux

Information in this post can still be considered valid but it has been updated in our official documentation.

AnuBitux is, first of all, a Debian based distro that aims to provide a simple and safe environment to use and manage cryptocurrencies. Its aim is to allow all the users to safeguard their cryptos and their privacy like pros, without the need to prepare any complex set-up. But it is not only a customized operative system, it also wants to be a place to share knowledge and best practices, providing tutorials based on AnuBitux that can be applied in every situation.

What is included in the AnuBitux OS?

As mentioned, AnuBitux is an environment based on Debian 11 Bullseye but with many customizations:

  • it includes a lot of useful tools to manage cryptocurrencies, like address generators and desktop wallets. Many tools are includes as AppImages, so that end-users will be able to check hashes by themselves;
  • the Firefox browser is improved to prevent fingerprinting and the storage of any kind of information on the device. It also has a lot of useful and interesting bookmarks to easily reach online resources;
  • if more privacy is neede, Tor Browser is also available;
  • AnonSurf has been added, to help users safeguard their privacy while accessing online resources;
  • Python tools run in dedicated virtual environments;
  • many printer drivers are installed. They could be useful to generate and print paper wallets completely offline;
  • it also includes many solutions to recover access to lost wallets;
  • and more...

The most significant part of AnuBitux is his collection of tools. Some of them have been selected for this environment, others have been developed by the AnuBitux team to fit the needs that users may encounter.

Tools list

The tools are placed in dedicated sub-menus, divided into different categories so that users will be able to quickly find what they need. Some tools did not provide a menu entry. In this case, AnuBitux has arranged a custom menu entry that also shows some useful information about the tool, like examples and suggestions.

Address generators

The first category is dedicated to the address generators, a collection of tools that could be used to securely generate crypto addresses to store, for instance, on a paper wallet.

  • BitAddress is a javascript tool that runs locally and creates a Bitcoin paper wallet and that can be found at bitaddress.org;
  • CashAddress is a javascript tool that runs locally and creates a Bitcoin Cash paper wallet and that can be found at cashaddress.org;
  • DashPaperWallet is a javascript tool that runs locally and creates a Dash paper wallet that can be found on paper.dash.org;
  • Dice2Seed is a tool that obtaines a mnemonic seed starting from the entropy obtained rolling a dice;
  • DiceTracker is a tool that generates printable paper wallets starting from 256 dice rolls, removing the need to trust software's entropy;
  • LastWord is a tool that finds a last word that respects the BIP39 checksum for mnemonic phrases;
  • LiteAddress is a javascript tool that runs locally and creates a Bitcoin paper wallet and that can be found at liteaddress.org;
  • MoneroWalletGenerator is an offline Monero wallet generator that can also be found on moneroaddress.org; 
  • WalletGenerator is a multi-currency paper wallet generator that supports dozens of different coins;
  • XMR AddressGen is a tool to generate XMR addresses that can also be reached on this website.

Encryption

Here there are tools that could be useful to interact with anything regarding encryption.

  • BitWarden is a very popular, powerful and open source password manager;
  • KeepassXC is an offline open source cross-platform password manager, that could also store any kind of information like mnemonic seed phrases;
  • Kleopatra manages PGP Keys and allows to encrypt, decrypt, sign and verify text and files;
  • VeraCrypt is a cross-platform open source tool that allows to encrypt and decrypt volumes and disks with many algorithms;
  • zuluCrypt is a simple,feature rich and powerful solution for hard drives encryption. More info here.

Hardware Wallets

This category encloses the official clients of some hardware wallets. Many of them are also supported by desktop clients like Electrum, since the udev rules have been set up in AnuBitux so that they can be connected without troubles. 

  • BC Vault is the official client to manage BCVault hardware wallets.
  • BitBoxApp is the official client to manage BitBox hardware wallets;
  • KeepKey is the official client for KeepKey hardware wallets;
  • Ledger Live is the official client to manage Ledger hardware wallets like the Nano S model and the Nano X model;
  • Setup udev rules is a script that sets up the udev rules, needed by the clients to interact with hardware wallets (this is done completely offline);
  • Trezor Suite is the official client to manage Trezor hardware wallets like the Model One and the Model T.

Recovery

In this section there is a collection of tool that could help to recover access to lost wallets or locked funds.

  • BTCKeySearch is a tool , developed by the AnuBitux team, that scans files in a directory and in its subdirectories, looking for Bitcoin private keys, Monero private spend keys and BIP32 root keys (xpriv);
  • BTCRecover is a powerful tool that provides many solutions to recover access to wallets, even to perform dictionary and bruteforce attacks;
  • BulkExtractor is a tool that rapidly scans directory and volumes for any kind of input like regular expressions;
  • HashCat is a very popular password cracking tool that supports a lot of different algorithms. Please refer to the official forum to learn how to use it in every specific case;
  • Photorec tries to recover deleted files or part of them and could be useful, for instance, for accidentally deleted wallets.
  • SeedSearch is a tool, developed by the AnuBitux team, that scans files in a directory and in its subdirectories, looking for BIP39 mnemonic seeds;
  • SeedRecover is part of the BTCrecover tool. It tries to fix a mnemonic seed phrase with mistakes and/or missing words;
  • SeedSearch is a tool, developed by the AnuBitux team, that scans files in a directory and in its subdirectories, looking for BIP39 mnemonic seeds;
  • VeraDecrypt is a tool, developed by the AnuBitux team, that tries to unlock VeraCrypt volumes and containers using a list of passwords given by the user.

UsefulTools

In this section there are some tools that could be useful in many AnuBitux's use cases.

  • FFMpeg is a complete, cross-platform solution to record, convert and stream audio and video. For this purpose, it could be used to record the desktop screen while doing operations that need to be reported to a third party. Never use it while generating sensible information like private keys or new wallets or, in general, for information that are not supposed to be shared;
  • Flameshot is a powerful tool to capture and edit screenshots
  • Password (passgen.py) is a secure password generator developed by the AnuBitux team;
  • QREncode is a tool that generates QR codes from strings and that could be very useful to store addresses and keys offline. More information here.
  • QRScan is a lightweight QR scanner with a terminal user interface (TUX) that works on files and throught the webcam;
  • QtQR is a GUI tool to generate and scan QR codes; 
  • TShark is a CLI tool for packet capturing;
  • Wireshark is the world’s foremost and widely-used network protocol analyzer. It could be used to capture network traffic during sensible operations and verify that only proper network traffic was exchanged.

Wallets

In this category there is a collection of many desktop wallets, so that the users, in most cases, will be able to use their coins without the need to install any client.

  • Coinomi is a multi-chain wallet that is widely used;
  • ElectronCash is a version of the well known Electrum wallet that works with Bitcoin Cash;
  • Electrum(LTC) is a version of the well known Electrum wallet that works with Litecoin;
  • Electrum(BTC) is one of the most known and used SPV Bitcoin wallets; 
  • Exodus is a multi-currency and multi-service desktop wallet;
  • FeatherWallet is a very complete and powerful Monero wallet with a comfortable user interface that is similar to Electrum's one;
  • GreenWallet is a desktop wallet, developed by Blockstream that supports Bitcoin and even L-BTC on the Liquid sidechain.
  • Monero-CLI is the official command line Monero wallet;
  • MyCrypto is a wallet that allows to manage Ethereum accounts;
  • MyMonero is a lightweight and efficient Monero wallet;
  • Sparrow wallet is a Bitcoin wallet caring for user's privacy;
  • ZecWallet Lite is a fully featured shielded wallet for Zcash;
  • MetaMask is a wallet for Ethereum accounts and a gateway to blockchain apps. It can be found as an extension of the FireFox browser;
  • Temple is a Tezos wallet that can be found as an extension of the FireFox browser.

Wallet Tools

In this section there are tools that can be used to manage wallet related stuff, especially mnemonic seed phrases.

  • BIP39 is the well known tool developed by Ian Coleman that handles derivation paths based on BIP39 wordlists. It is also available on his website;
  • Coinb.in is a tool that allows users to sign, verify and broadcast transactions that could also have been written from scratch. The tool is also available on the web at the coinb.in page.
  • SeedCheck.py is a tool developed by the AnuBitux team that tries to find out the correct derivation path or the right software a seed was used with;
  • XMR Address Test is a tool related to Cryptonote Addresses, used by the Monero protocol;
  • XMR check tx is a tool that can be used to verify XMR transactions, providing the necessary keys;
  • XMR message sign can be used to sign and verify messages using XMR keys;
  • Xpub-converter is a tool, by Jameson Lopp, that allows to convert among many different formats of xpub keys.
