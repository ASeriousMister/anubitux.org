AnuBitux 3.4

Release date: 06-Jul-2024

Versions involved: live amd64

Changes:

  • tools and distro updated;
  • new tool BlindMnemonic by the AnuBitux team, that allow two users to generate portions of a mnemonic seed and some related deposit addresses;

AnuBitux 3.3

Release date: 11-May-2024

Versions involved: live amd64

Changes:

  • Distro updated with debian updates;
  • Tools updated;
  • New Tools by the AnuBitux Team: BlindGen, a tool that allows two operators to generate two parts of a private key and a public address where to deposit funds, so that nobody has access to the whole private key;
  • Dependency issues of SeedCheck have finally been fixed;
  • Most of the tools have now a GUI, so that user does not have to write commands with a complicated sintax;
  • New password generator with enhanced features, like copying to clipboard and empting the clipboard.
