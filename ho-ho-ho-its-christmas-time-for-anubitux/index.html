HO HO HO! It's Christmas time for AnuBitux!

December 1, 2022 AnuBitux

Check out this new customized AnuBitux! A very cool Christmas theme with some gifts! New tools and updates in it!

The distro has received Kubuntu's updates. Also Tor Browser and many wallet softwares have been updated.

The BitWarden password manager has been added, so you can manage more passwords in your working environment.

And, last but not least, a new tool by the AnuBitux team! Try to type password in your terminal and a new secure password will be generated every time you'll need it. Do not forget to store it somewhere if you used it. AnuBitux is not going to store anything, as always. Your security lies only on you!

Grab your Christmas version of AnuBitux! It will be available only for a limited time!

Merry Christmas from the AnuBitux team!!!
